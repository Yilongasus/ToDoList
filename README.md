# ToDoList 使用說明

<hr>

## 基本功能
### 一新增任務
#### 在請輸入任務名稱輸入框中，輸入任務名稱，再按下右方+咖啡色按鈕，即完成新增任務。
<br>
<br>
### 一查看任務列表 
#### 清單分為未完成、已完成、所有共三個分頁。
#### ◆ 未完成：顯示未勾選的任務。
#### ◆ 已完成：顯示勾選的任務，任務會呈現刪除線。
#### ◆ 所有：顯示所有的任務。
<br>
<br>
### 一更新任務狀態
#### 勾選會取消勾選核取方塊，即可更新任務狀態。
<br>
<br>
### 一刪除任務 
#### ◆ 刪除單一任務：按下某任務右方X紅色按鈕，即可刪除此項任務。
#### ◆ 刪除全部任務：按下清單左下方"刪除全部任務"紅色按鈕，即可刪除所有任務。
